Privacy Policy

Last updated: 2 June 2026

MIQ is a free, open-source macOS app. This website and the app set no cookies and use no analytics or tracking. The only personal data involved is described below.

Website

This site is hosted on GitHub Pages. When you visit, GitHub Inc. (USA) automatically records technical data such as your IP address and browser type in its server logs, and the embedded screenshots and demo video are loaded from GitHub servers. This processing is carried out by GitHub under its privacy statement.

App

The MIQ app runs entirely on your Mac. It does not transmit your files or any usage data. It only contacts GitHub to check for new releases; this request reveals your IP address to GitHub, as with any web request.

Your rights

Under the Swiss Federal Act on Data Protection (revFADP) you may request access to, correction of, or deletion of personal data concerning you. As this project holds no personal data of its own, such requests relate to the logs held by GitHub. For anything regarding this project, contact the operator below.
